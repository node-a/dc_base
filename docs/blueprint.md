# **App Name**: Aqua Access

## Core Features:

- Account Creation: Securely create a new account with first name, last name, email, and password.
- Login Authentication: Authenticate users via email and password for secure access.
- Session Management: Maintain user sessions to keep users logged in, utilizing secure tokens and cookies.

## Style Guidelines:

- Background color: Very light grayish-blue (#F0F4F8) to give a clean, spacious feel reminiscent of Apple's design.
- Primary color: Subtle light blue (#A8D0E6) for interactive elements like buttons and links.
- Accent color: Light cyan (#95E1D3) for secondary interactive elements.
- Font: 'Inter', a sans-serif font, known for its clean and modern appearance, for both headings and body text.
- Minimalist layout with ample white space to emphasize content clarity and a modern, uncluttered user interface, similar to Apple's design philosophy.
- Use simple, outline-style icons for any supplementary UI elements.
- Subtle transition animations for a smooth, intuitive user experience, enhancing the minimalist design without being distracting.
